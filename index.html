Featured Projects (The Highlights)

BioMind-DR: Biomedical Metacognitive Intelligence

The Magnum Opus. A system that mimics the human brain to cure diseases.

Role: Project Leader & Lead Researcher (1 Year)

The Problem: Traditional drug repurposing is slow and disconnected. We needed a system that doesn't just "search" but "thinks" and verifies hypotheses like a scientist.

The Solution: Built a Leader-Expert-Council Multi-Agent System based on Metacognitive Memory Architecture (simulating Hippocampus & Neocortex).

  • Fast System (Hippocampus): L0 Working Memory (Redis) & L1 Episodic Buffer (Kafka) for real-time context and event streaming.
  • Slow System (Neocortex): L2 Knowledge Core powered by KuzuDB.
  • Bi-temporal Graph: Tracks validity of knowledge over history.
  • Evidence Graph: Isolates conflicting data for the Council Agent to adjudicate.
  • Memory Consolidation: L3 Strategy (MariaDB) & L4 Long-term Storage (MinIO) allow the system to learn from past inferences.

Impact & Result:

  • Outperformed SOTA models: Achieved 48.2% on HLE and 94.7% on MRCR v2, beating Gemini 3.0 Pro and GPT-5.1 in biomedical reasoning tasks.
  • Integrated a virtual dry-lab for hypothesis verification.

Tech Stack: Python, PyTorch, KuzuDB, Kafka, Redis, MariaDB, MinIO.
